# 🏥 Medicare Pro – Configurable Healthcare Website Template

![React](https://img.shields.io/badge/React-19-blue?logo=react)
![Vite](https://img.shields.io/badge/Vite-8-purple?logo=vite)
![TailwindCSS](https://img.shields.io/badge/TailwindCSS-v4-38BDF8?logo=tailwindcss)
![License](https://img.shields.io/badge/Status-Open%20Source-success)

A modern, responsive, and configurable healthcare website template built using **React**, **Vite**, and **Tailwind CSS**.

Designed for hospitals, clinics, diagnostic centers, and healthcare organizations, this project allows easy customization through a centralized configuration file without modifying the UI components.

---

## ✨ Features

- 🏥 Modern Healthcare Landing Page
- 🎨 Fully Customizable Theme
- 👨‍⚕️ Doctors & Specialists Section
- 🩺 Multiple Medical Departments
- 📅 Appointment Booking Form
- 💬 WhatsApp Integration
- 📍 Google Maps Integration
- ⭐ Patient Testimonials
- 🖼️ Gallery Section
- ❓ FAQ Section
- 📱 Fully Responsive Design
- ⚡ Fast Performance with Vite
- 🔍 SEO-Friendly Configuration

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/JyotiKumari-web8/medicare-pro.git
```

### Go to Project Folder

```bash
cd medicare-pro
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

---

## 🎨 Customization

The website is configuration-driven.

You can easily customize:

- Hospital Name
- Logo
- Theme Colors
- Contact Information
- Doctors
- Departments
- Emergency Numbers
- Google Maps
- WhatsApp Number
- Gallery
- Testimonials
- FAQ

Simply edit:

```text
src/hospitalConfig.js
```

No need to modify the React components.
